Project Title: Analyzing Game Sales Across Regions and Platforms

Description:
This project analyzes game sales data across different platforms and regions using SQL.
The analysis includes inserting new data, updating records, deleting records, and using
aggregate functions and joins to extract meaningful insights.

Files Included:
1. Games.csv
2. Game Sales.csv
3. game_sales.sql
4. README.txt

Key Operations:
- Data insertion, update, and deletion
- Aggregate functions (SUM)
- INNER JOIN, LEFT JOIN
- Filtering and duplicate handling

Tools Used:
- SQL (MySQL)
